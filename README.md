# Portfolio: Digitalization of the Grant Application Process 🚀

**Author:** Ruslan Masharau  
**Target Role:** Kaufmann für Digitalisierungsmanagement (IHK) / Process Officer / Junior IT PMO  
**Status:** In Progress (Sprints 1-5)  

## 📌 Executive Summary
This repository contains a comprehensive, end-to-end project portfolio demonstrating the digitalization of a manual grant application process for a fictional NGO, **"Gemeinsam Stark e.V."** (Munich, 8 employees). 

The objective is to showcase practical, IHK-compliant competencies in **Requirements Engineering, Business Process Modeling (BPMN 2.0), and Low-Code/No-Code Automation**, bridging the gap between business needs and IT infrastructure.

**Business Value:**
* **As-Is:** Manual PDF/Email-based application handling, high processing time, unstructured data.
* **To-Be:** Automated workflow, centralized data architecture, reduced cycle time by an estimated 40%.

## 🛠 Tech Stack & Methodologies
* **Process Modeling:** BPMN 2.0 (Camunda Modeler), Draw.io
* **Agile / PMO:** Scrum framework, Jira (Backlog/Sprints), Confluence (Documentation)
* **Architecture / Design:** DFD, ERD, REST API principles, ADR (Architecture Decision Records)
* **Automation (Low-Code):** n8n / Make / Dify (AI-Agent workflows)
* **Compliance:** GDPR / DSGVO standards
